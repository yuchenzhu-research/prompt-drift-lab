## Prompt A / Prompt B 的设计取舍依据

本项目采用 Prompt B 作为主实验提示词，并非基于主观偏好，
而是基于一份独立的对照评测报告所揭示的系统性差异。

详见：
`06_方法论补充与对照依据/Prompt_A_B_三段式转译器_对照评测报告.pdf`

该报告基于 IFEval / IHEval / MT-Bench / G-Eval 等评测思想，
从以下维度对 Prompt A（原版）与 Prompt B（加强版）进行了对照：

- 硬性指令遵循（结构、角色、负向约束）
- 指令层级与冲突处理能力
- 三段式组件级评分（事实快照 / 搜索指令 / 深挖指令）
- 去偏与信息密度校准

结论显示：Prompt B 在机制类问题（Q3、Q4）上具有显著优势，
且该优势来源于可解释的设计差异，而非输出冗长。
