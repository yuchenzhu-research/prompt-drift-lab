# 07 Deep Research 搜索汇总

本文件夹收录与 **Prompt Drift Lab** 相关的 Deep Research 检索与综述材料，用于提供外部背景、术语对齐与证据锚点，帮助学术与工业读者快速定位本项目在相关研究与工程实践中的位置。

## 适用范围

- **背景与动机**：prompt drift 的研究问题、常见失效形态与研究趋势概览  
- **方法论对齐**：评测设计、指标维度、可复现协议的外部参照  
- **证据链补充**：提示词 A/B 选择依据、稳定性与工程化取舍的旁证材料  

## 使用规范

- **不替代实验结论**：本文件夹材料仅作为背景与引用锚点；项目数值结论以仓库 `runs/`、`summary.csv`、逐题评分与原始输出为准。  
- **引用建议**：正式写作中优先引用材料中所指向的原始论文/官方文档；本文件夹 PDF 可作为“检索汇总/二次整理”的辅助来源。
