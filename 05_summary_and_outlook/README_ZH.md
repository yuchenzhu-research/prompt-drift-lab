# 总结与展望（Prompt Drift Lab · Summary & Outlook）

> 本文档用于 **总结已完成的实验现象、澄清研究边界，并给出不超出现有证据的后续研究方向**。
>
> **重要更正说明**：本项目当前版本 **未实现任何自动化扰动生成器（perturb）或大规模 prompt 扫描机制**。以下内容已严格对齐仓库中**实际完成并落盘的实验资产**，不包含任何未实施的方法、工具或系统性扩展。

---

## 1. 本项目已经完成了什么

本项目围绕 **Prompt Drift** —— 即在任务保持不变的前提下，因提示词在格式、措辞或约束表达上发生微小变化，而导致的大语言模型在**指令遵循、结构稳定性与语义一致性**方面出现系统性偏移 —— 构建了一个**可复现的最小实验闭环**。

具体而言：

- **输入侧（Prompt Design）**：在固定任务与题集条件下，人工构造少量、可解释的提示词差分版本，用于有针对性地暴露指令遵循与结构稳定性问题。
- **输出侧（Model Outputs）**：完整保留不同模型在不同提示词条件下的原始输出（PDF），并在此基础上形成结构化的结果汇总（CSV）。
- **评测侧（Evaluation）**：采用跨模型互评作为主方法，并以模型自评作为辅助对照，配套明确的评测协议与评测提示词（详见 `03_evaluation/`）。

> **边界声明**：本项目中观察到的所有 prompt drift 现象，**均来自人工设计的提示词版本差异**；当前版本未使用自动扰动、随机变换或大规模 prompt 枚举。

---

## 2. 主实验范围与方法取舍

为避免读者对实验结论产生误解，仓库中对实验范围与用途作了明确区分：

- **主实验（可复现）**：围绕 `Prompt B` 及其人工构造的变体（`long` / `weak` / `conflict`）进行对照评测与统计汇总。
- **先导探索（Pilot）**：`Prompt A` 仅用于早期探索与问题暴露，不参与任何主实验的定量比较或统计结论。

**方法取舍说明**：选择 `Prompt B` 作为主实验基线，并非随意决策，而是基于先导阶段中 A/B 在**可执行性、结构稳定性与工程可复核性**方面的对照结果。相关依据与复核路径已在 `06_` 目录中集中记录。

---

## 3. 基于已完成实验的关键观察

结合 `summary.csv` 及各类分组统计结果，本项目当前版本能够支持以下**不超出现有证据范围**的观察：

1. **即使是数量极少、人工设计的提示词差分，也足以触发显著的指令遵循率与结构稳定性变化**。
2. **提示词更长或更详细，并不必然带来更高的稳定性**；在部分条件下反而引入约束稀释或执行回退。
3. **对约束语气的弱化（例如将 must 替换为 should）会明显提高 drift 发生概率**。
4. **包含指令张力或潜在冲突的版本，更容易暴露模型的隐式优先级选择行为**。

这些观察均来自已完成的实验结果汇总，而非对未来系统性扫描的推断。

---

## 4. 外部研究的作用（解释视角，而非方法补充）

本节仅用于将本项目中已观察到的现象，与现有研究中的相关概念进行对齐，
**并不意味着本项目已实现、复现或比较以下研究方法**：

- prompt brittleness / prompt sensitivity
- instruction-following stability
- format robustness

这些研究为理解以下现象提供了背景视角：
即便在输入语义几乎不变的情况下，LLM 的输出行为仍可能因极小的提示词变化而产生明显漂移。

---

## 5. 方法局限与未来工作（严格区分）

### 5.1 当前版本的明确局限

- 提示词差分完全依赖人工设计，覆盖空间有限。
- 未引入自动扰动、随机化或系统性扫描机制。
- 实验结论仅适用于当前提示词空间与评测协议设定。

### 5.2 未来可能的扩展方向（未实施）

> 以下内容仅作为**潜在后续研究方向的提示**，不构成本项目当前版本的方法组成部分：

- 若未来引入自动化扰动生成或更大规模的 prompt 扫描，应作为**独立版本或独立实验报告**发布，
  且不得与当前版本的定量结论混用。

---

## 6. 项目定位总结

本项目的核心价值在于：

- **将 prompt 变化视为实验自变量，而非调参技巧**；
- **通过少量、可解释的对照实验，明确展示 prompt drift 的存在形式与类型**；
- **借助严格的文件结构与评测协议，使所有结论均可被复核与复现**。

因此，Prompt Drift Lab 并非一个“大而全”的 benchmark，而是一个**研究边界清晰、证据诚实的最小研究工程**。
