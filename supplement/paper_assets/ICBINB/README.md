# Paper Assets

This directory contains the figures and tables used in the paper.

All assets are derived from the released evaluation results and are provided here
for ease of inspection and reference. They correspond directly to the results
reported in the main paper.

## Figures

- `figures/fig_invalid_taxonomy_distribution.(png|pdf)`

## Tables
- `tables/table_main_summary.(csv|tex)`
- `tables/table_main_by_question.(csv|tex)`
- Additional tidy CSV files are included where applicable.

These files are intended for reading and verification. They are not paper source
files and are not intended to support rebuilding the manuscript.
