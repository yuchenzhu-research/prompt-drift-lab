# 实验设计总览

本目录定义了 Prompt Drift Lab 的整体实验协议，
用于分析在任务、输出结构与评测规则固定的前提下，
提示词微小变化对模型输出行为的影响。

## 实验目标

- 系统观察提示词结构变化引发的：
  - 格式遵循率下降
  - 输出结构崩坏
  - 语义偏移与指令漂移
- 为后续机制分析与设计启示提供可复现实验基础

## 实验变量说明

- 自变量：
  - 提示词版本（见 `02_提示词版本/`）
- 控制变量：
  - 问题集（`问题集.jsonl`）
  - 标准输出结构（`标准输出结构.md`）
  - 评测规则（`03_评测规则/`）
  - 模型与参数（见 `protocol.yaml`）

## 本目录文件说明

- `实验设计_五步法.md`：整体实验流程与步骤拆解
- `protocol.yaml`：模型、参数与运行配置（实验协议）
- `标准输出结构.md`：所有实验条件共享的输出模板
- `问题集.jsonl`：固定问题集
