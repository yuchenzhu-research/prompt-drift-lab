# 评测协议（Evaluation Protocol）

## 评测目标
比较不同提示词条件下，模型在结构遵循与指令执行行为上的稳定性差异。

## 评测对象
- Prompt A / Prompt B
- 固定问题集（4 题）

## 评测方式
- 硬性结构合规：规则判定（通过 / 失败）
- 行为表现评分：基于量表的相对评分

## 评测执行
- 评测由 LLM 自动执行（Gemini），
  按照既定评测 Prompt 与评分量表进行
- Gemini 仅作为评测执行器，不作为事实裁判

## 偏差控制
- 盲评：不暴露 Prompt A / B 标签
- 长度控制：避免“更长=更高分”的偏差
